# おきなわGOLFなび

沖縄県内のゴルフ大会、募集要項、成績、ゴルフ場、練習場、イベント情報をまとめる地域情報サイトの初期プロトタイプです。

## ファイル構成

- `index.html` サイト本文、SEOメタ情報、初期掲載データ
- `articles/` ブログ・読み物記事ページ
- `styles.css` レイアウト、配色、レスポンシブ対応
- `script.js` キーワード検索と大会カテゴリ絞り込み

## 想定する運用

- 大会情報は「開催予定」「募集中」「成績あり」「ジュニア」などの状態で管理
- トピック（最新情報）は大会結果、募集開始、イベント開催、施設更新などを3件から5件程度に絞って掲載
- ブログ記事は初心者、観光ゴルフ、大会参加、ジュニア育成など検索意図ごとに作成
- ゴルフ場・練習場は個別ページ化し、住所、公式サイト、料金、営業時間、予約導線を掲載
- 広告は本文途中の横長枠、施設ページの関連枠、パートナー紹介枠を中心に配置
- 更新日は各記事・施設ページごとに明記
- 公式発表、主催者ページ、各施設公式サイトを一次情報として確認

## 公開前に追加したい内容

- 沖縄県内の全ゴルフ場の正確な公式URL、住所、電話番号
- 練習場の屋内・屋外分類、営業時間、料金、レッスン有無
- 大会ごとの募集要項PDF、成績PDF、申込締切
- パートナー問い合わせフォーム
- Google AdSenseなど広告タグの導入
- Google Search Console / Analytics設定
