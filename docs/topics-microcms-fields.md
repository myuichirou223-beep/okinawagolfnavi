# topics API field plan

おきなわGOLFなびの「トピック（最新情報）」をmicroCMSで管理するためのフィールド案です。

## API設定

- API名: トピック
- エンドポイント: `topics`
- 形式: リスト形式

| 表示名 | フィールドID | 種類 | 必須 | 用途 |
| --- | --- | --- | --- | --- |
| タイトル | title | テキストフィールド | yes | トピックの見出し |
| カテゴリ | category | セレクト | yes | 大会結果 / 募集開始 / イベント / 試打会 / お知らせ / ジュニア |
| 説明文 | description | テキストエリア | yes | 一覧に出す短い本文 |
| 公開日 | published | 日付 | yes | 表示順と更新日表示 |
| 掲載状態 | status | セレクト | yes | draft / published / archived |
| リンクURL | linkUrl | テキストフィールド | no | 公式ページや記事への外部リンク |
| 検索キーワード | tags | テキストフィールド | no | サイト内検索用の補助語 |
| 画像 | image | 画像 | no | 将来の拡張用。現在のトップ最新情報リストでは使用しない |

## 運用ルール

- 新しい情報ほど上に表示されるよう、`published` を更新日として入れる。
- `status` は公開するものだけ `published` にする。
- 外部リンクがない場合、リンクURLは空でよい。
- トップページには最新5件を表示する。
- 表示はカテゴリ、公開日、タイトル、説明文、詳細リンクのリスト形式にする。

## 初期データの一括登録

`microcms/imports/topics-import-template.csv` を用意しています。
microCMSのトピックAPI画面でCSVインポートを開き、このファイルをアップロードしてください。

画像フィールドは空欄にしています。現在のトップページでは画像を使わないため、必要になったタイミングで登録してください。
