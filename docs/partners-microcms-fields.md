# partners API field plan

おきなわGOLFなびのパートナーロゴスライダーをmicroCMSで管理するためのフィールド案です。

## API設定

- API名: パートナー
- エンドポイント: `partners`
- 形式: リスト形式

| 表示名 | フィールドID | 種類 | 必須 | 用途 |
| --- | --- | --- | --- | --- |
| パートナー名 | name | テキストフィールド | yes | ロゴ画像の代替テキスト。画像がない場合は文字ロゴとして表示 |
| ロゴ画像 | logo | 画像 | no | スライダーに表示するロゴ |
| 公式URL | websiteUrl | テキストフィールド | no | ロゴクリック時のリンク先 |
| 掲載状態 | status | セレクト | yes | published / draft / archived |
| 表示順 | displayOrder | 数字 | no | 小さい数字ほど先に表示 |

## 運用ルール

- 公開するものは `status` を `published` にする。
- ロゴ画像は横長推奨。背景透過PNGまたは白背景ロゴが扱いやすい。
- `websiteUrl` が空欄の場合、クリックできないロゴとして表示する。
- ロゴ画像が未登録の場合、`name` を文字ロゴとして表示する。
