# tournaments API field plan

おきなわGOLFなびの「大会情報」と「年間スケジュール」をmicroCMSで管理するためのフィールド案です。

## API設定

- API名: 大会
- エンドポイント: `tournaments`
- 形式: リスト形式

| 表示名 | フィールドID | 種類 | 必須 | 用途 |
| --- | --- | --- | --- | --- |
| 大会名 | title | テキストフィールド | yes | 大会カード・年間スケジュールの見出し |
| 開催月 | month | テキストフィールド | yes | 例: 1月 / 3月 / 春 / 夏 |
| 開催日 | dateLabel | テキストフィールド | no | 例: 2026年3月5日-8日 |
| 会場 | venue | テキストフィールド | no | ゴルフ場名、県内会場など |
| 地域 | area | テキストフィールド | no | 南城市、沖縄県内など |
| 種別 | category | セレクト | no | 一般 / シニア / 女性 / ジュニア / プロ |
| ステータス | status | セレクト | yes | 募集中 / 開催予定 / 成績あり / 確認中 / draft / archived |
| 説明文 | description | テキストエリア | no | 大会カードの短い説明 |
| 募集要項URL | entryUrl | テキストフィールド | no | 募集中・開催予定の導線 |
| 大会概要URL | overviewUrl | テキストフィールド | no | 大会概要ページへの導線 |
| 成績URL | resultUrl | テキストフィールド | no | 成績ありの導線 |
| 公式URL | officialUrl | テキストフィールド | no | 公式情報の導線 |
| 表示順 | displayOrder | 数字 | yes | 年間スケジュールの並び順。10, 20, 30... |
| 検索キーワード | tags | テキストフィールド | no | サイト内検索用の補助語 |

## 運用ルール

- 年間スケジュールは `displayOrder` の小さい順に表示する。
- トップの大会セクションは年間スケジュール型で表示する。大会カード型の重複表示は使わない。
- 絞り込みは `category` の「一般 / シニア / 女性 / ジュニア / プロ」と、`status` の「開催済み / これから開催」で行う。
- 各スケジュール行には「大会公式ページ」「大会概要」「成績表」の3ボタンを表示する。
- `officialUrl` があれば「大会公式ページ」ボタンが有効になる。
- `overviewUrl` があれば「大会概要」ボタンが有効になる。未入力の場合は `entryUrl` を代替表示する。
- `resultUrl` があれば「成績表」ボタンが有効になる。
- 下書きにしたい場合は `status` を `draft` にする。公開ページには出さない。
- URLが未確認の場合は空欄でよい。
